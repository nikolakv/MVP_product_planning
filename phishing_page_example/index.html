Important: Buttons are disabled and form-action 'none' in CSP blocks submission.

Common phishing red flags shown here

  • Mismatched URL (domain doesn’t match the brand)
  • Urgency (“immediate action required”)
  • Shortened links that obscure the destination (don’t click unknown short links)
  • Requests for passwords or MFA codes via webforms or email

Learn more about phishing awareness and simulations from CISA and SANS.

What to do if this were real

  1. Close the tab. Do not enter credentials.
  2. Report the message/link to your security team.
  3. If you entered info by mistake, change your password and report it immediately.

This page sets noindex,nofollow to avoid appearing in search results, and a restrictive Content-Security-Policy delivered via meta to block scripts and form submissions.
